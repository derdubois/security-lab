Devices have two means of identification:
1. An IP address
2. A Media Access Control (MAC) address (serial number)

IP address (Internet Protocol) can be used to idetnify the a host on a network for a period of time.

An IP address is set of number that are divided into four octets. 

A public address is used to indetify the device on the Internet, whereas a private address is used to indetify a device amongst other devices.

Public IP address are given by your Internet Service Provider (ISP) at a monthly fee. (IPv4 - 2^32 IP addresses)

IPv6 supports up to 2^128 of IP addresses 


MAC addresses
Devices on a network will all have a physical network interfacem which is a microchip board found on the device's motherboard. 

This network interface is assigned a unique address at the factory it was built at, called a MAC (Media Access Control ) address.

The MAC address is a twelve-character hexadecimal number split into two's and separated by a colon. The first sisx characters repsent the company that made the network interface, and the last six is a unique number. (a4:c3:f0:85:ac:2d)

MAC address can be "spoofed", the spoofing occurs when a networked device pretends to identify as another using its MAC adress.
